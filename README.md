# JsonParser

![C++](https://img.shields.io/badge/language-C%2B%2B-blue.svg)

hand-made json parser, made with love by literal kernel.

* lexer  - scaner
* parser - recursive-descent parsing

test

* lexer  - lexer tester
* tryrun - test if parser work
* data.json - test data

# build

to build is simple, just use CMake

```sh
$ cmake -S . -B ./build
```

then start build

```sh
$ cmake --build ./build
```
